Detailed Description

template<typename T>
class xrn::util::BasicClock< T >

Simplifies time measures.

<Util/Clock.hpp>

::xrn::util::Clock provide an high resolution clock allowing precise elapsed time measures.

Usage example:

::xrn::Clock clock;
...
::xrn::Time time1{ clock.getElapsedTime() };
...
::xrn::Time time2{ clock.restart() };
